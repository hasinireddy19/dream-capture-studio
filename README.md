# Dream Capture Studio

> The official portfolio website for Dream Capture Studio. An elegant, fully responsive photography website featuring an interactive 35mm film strip, a dynamic masonry gallery, and a custom booking calculator.

## 🌟 Features

- **Responsive Design**: Flawless display across all devices (mobile, tablet, and desktop).
- **Interactive 35mm Film Strip**: A smooth, infinitely scrolling animated film strip that updates the hero image on hover or tap.
- **Dynamic Masonry Gallery**: Beautifully structured portfolio section with smooth category filtering (Weddings, Birthdays, Fashion).
- **Lightbox Image Viewer**: Click on any gallery image to view it in an immersive, full-screen lightbox.
- **Booking Calculator**: An interactive pricing calculator for clients to estimate the cost of their photography session based on hours and add-ons.
- **Custom Animations**: Elegant CSS-based fade-in effects and smooth transitions for a premium, airy aesthetic.

## 🛠️ Technologies Used

- **HTML5**: Semantic and accessible page structure.
- **CSS3**: Vanilla CSS with CSS Variables for easy theming, flexbox/grid for layouts, and keyframes for animations.
- **JavaScript (Vanilla)**: DOM manipulation, event listeners, intersection observers for scroll animations, and interactive component logic.

## 🚀 Getting Started

1. Clone this repository to your local machine:
   ```bash
   git clone https://github.com/your-username/dream-capture-studio.git
   ```
2. Navigate into the project directory:
   ```bash
   cd dream-capture-studio
   ```
3. Open `index.html` in your favorite web browser to view the website. No build steps or local servers are strictly required!
